# 1. 서론

## 1.1 문서 목적 및 범위

본 문서는 클라우드 기반 파일 저장 및 공유 시스템에 대한 테스트 수행 계획과 테스트 케이스를 정의하고 예상 결과를 기록하기 위해 작성되었다.

본 문서에서는 요구사항 정의서에 명시된 기능 및 품질 요구사항을 기준으로 테스트 범위, 테스트 항목, 테스트 케이스 및 예상 결과를 기술한다.

---

## 1.2 프로젝트 개요

본 시스템은 사용자가 파일을 업로드, 다운로드, 검색 및 공유할 수 있는 웹 기반 파일 관리 시스템이다.

또한 사용자 인증 기능과 접근 권한 제어 기능을 제공하여 안전한 파일 공유 환경을 제공한다.

---

## 1.3 용어 정의

| 용어   | 설명                         |
| ---- | -------------------------- |
| 사용자  | 시스템을 사용하는 일반 회원            |
| 관리자  | 시스템 관리 권한을 가진 사용자          |
| 업로드  | 사용자의 파일을 서버에 저장하는 기능       |
| 다운로드 | 서버 파일을 사용자 장치로 저장하는 기능     |
| 공유   | 다른 사용자에게 파일 접근 권한을 부여하는 기능 |

---

## 1.4 참조 문서

1. 프로젝트 정의서
2. 프로젝트 관리 계획서
3. 요구사항 정의서
4. 품질 요소 추정서

# 2. 테스트 개요

## 2.1 테스트 범위

테스트는 블랙박스 테스트를 기반으로 수행한다.

| 범위     | 내용            |
| ------ | ------------- |
| 사용자 기능 | 회원가입 및 로그인    |
| 파일 관리  | 업로드, 다운로드, 검색 |
| 파일 공유  | 공유 및 권한 설정    |
| 보안 기능  | 접근 제어         |
| 성능     | 응답 속도 확인      |

---

## 2.2 테스트 항목 및 통과 기준

| 기능별 테스트 항목 | 통과 기준             |
| ---------- | ----------------- |
| 회원가입 기능    | 정상 정보 입력 시 가입 성공  |
| 로그인 기능     | 올바른 계정 정보로 로그인 성공 |
| 업로드 기능     | 파일이 정상 저장됨        |
| 다운로드 기능    | 파일이 정상 다운로드됨      |
| 검색 기능      | 파일 검색 결과가 정확히 출력됨 |
| 공유 기능      | 지정 사용자에게만 공유 가능   |

| 비기능 테스트 항목 | 통과 기준         |
| ---------- | ------------- |
| 성능 테스트     | 업로드 시작 3초 이내  |
| 검색 테스트     | 검색 결과 2초 이내   |
| 보안 테스트     | 비인가 사용자 접근 차단 |
